## Como executar os projetos

Cada pasta funciona de forma independente. Entre no projeto desejado e instale as dependências.

### React Native / Expo

```bash
cd reactNative/NOME_DO_PROJETO
npm install
npm start
```

Comandos comuns disponíveis:

- `npm run android`
- `npm run ios`
- `npm run web`

### Vue.js / Vite

```bash
cd vuejs/NOME_DO_PROJETO
npm install
npm run dev
```

Outros comandos comuns:

- `npm run build`
- `npm run preview`
